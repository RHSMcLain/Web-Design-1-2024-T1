General Lists

Lists have an outer tag either ul or ol. And each item in the list is wrapped in an li tag.

  • Portland Trailblazers
  • Utah Jazz
  • Los Angeles Lakers
  • Seattle Supersonics (once and future)

In list items, it is OK to have other elements. For instance, you can include links or images -- or even other lists (we'll go into that more later).

Ordered Lists

Ordered lists start with <ol> tags, and give you a list with numbers (or some other sequencer depending on how you style it. It can be roman numerals, letters....

  1. Apples
  2. Bananas
  3. Cranberries
  4. Dates
  5. Elephant Ears
  1. Apples
  2. Bananas
  3. Cranberries
  4. Dates
  5. Elephant Ears

Definition lists:

Definition lists allow you to specify the word or phrase you're defining followed by the definition.

The structure is a <dl> tag. for each term you do a <dt> element with the term in it and a <dd> with the description or definition
