Tables

A table is designed for showing related data. It's kind of like a spreadsheet. Typically, rows across show different information about the same thing (like address information about a person. Columns going down show the same kind of information about different things. So zip code for each person.

A table has a table element on the outside. With that, there are tr elements (rows). Each row has a collection of th (table heading) or td (table data) elements.

You can also add sections to the table. You can have a thead, a tbody, and a tfoot. Usually the thead is for column titles and explanations. The tbody is for the data itself, and the tfoot is for summary information.
